Fun Facts

Adidas

The founder of Adidas, Adolf “Adi” Dassler, initially had no intention of starting a sports shoes company. In fact, Adi Dassler had completed a baking apprenticeship prior to starting the Company.

Microsoft Windows

Windows OS would have been called 'Interface Manager'. Gates had planned to release it under the same name.However, 'Windows' name prevailed because it best describes the boxes or computing 'windows' that were fundamental to the new operating system.

Google

Google is perhaps the biggest and most public spelling mistake ever made. Page and Brin settled on “Googol” as the name for their search engine, referring to the impossibly large number of the same name (the number 1 followed by a hundred zeros).When it came time to register the domain name they had made a mistake but stil went ahead with it.
